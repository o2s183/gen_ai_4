# gen_ai

This repository contains a LangChain-based notebook for loading documents, splitting text, creating embeddings, and running similarity search.

## Setup

1. Create a Python virtual environment (recommended):
   ```powershell
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1
   python -m pip install -r requirements.txt
   ```

2. Run the notebook `cppRAG.ipynb` in Jupyter.

## Files

- `cppRAG.ipynb`: main analysis notebook
- `requirements.txt`: Python dependencies
- `cppData.txt`: sample text source
